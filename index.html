Pug SASS/SCSS Starter Pack

Here's some simple boilerplate to get you started quickly!

This project utilizes pug and node-sass to spin up HTML pages with ease and speed. The power of easy-to-use templating and compiled CSS, ready-to-go!

How to Get Started:

In a terminal, clone the project and execute 'npm install' and then 'npm run dev' and enjoy!

Individual compilation

As inspection of the package.json will reveal, there are two separate scripts defined and these are run simultaneously using concurrently.

Compiling Pug Individually

Pug files can be compiled into HTML without watching for any changes on styles.scss by executing 'npm run pug' instead of dev.

Compiling SCSS Individually

Similarly to pug, the same can be done for SASS/SCSS. This command is 'npm run style' and it will only watch for changes to styles.scss without worrying about and pug files.
